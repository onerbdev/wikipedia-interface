JAVASCRIPT

O JavaScript é uma linguagem de programação interpretada, leve e voltada para o desenvolvimento web.

O JavaScript forma o combo web com outras duas tecnologias, são elas: o HTML e o CSS. Seguindo a analogia de que o HTML é o esqueleto, o CSS é a pele e o JavaScript os músculos e orgãos do corpo.

    Cada um com a sua própria função:

  • html: usado para criar a estrutura do site.
  • css: adiciona cores, fontes de texto, espaçamentos, posicionamentos e etc...
  • javascript: adiciona interatividade e movimento, permitindo que o usuário possa interagir de forma eficiente com o site.

Qual a sua função ?

Tornar páginas web interativas e dinâmicas (ex: cliques, animações, validação de formulários e etc).
