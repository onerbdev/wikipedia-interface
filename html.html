HTML

HTML é uma linguagem de marcação de texto, usada para criar a estrutura dos sites e web pages.

De forma simples, podemos usar a analogia de que o html é "esqueleto" de um site.

O html forma o combo web com outras duas tecnologias, são elas: o CSS3 e o JAVASCRIPT. Seguindo a analogia de que o HTML5 é o esqueleto, o CSS3 seria a pele e o JAVASCRIPT os músculos e orgãos do corpo humano.

    Cada um com a sua própria função:

  • html: usado para criar a estrutura do site.
  • css: adiciona cores, fontes de texto, espaçamentos, posicionamentos e etc...
  • javascript: adiciona interatividade e movimento, permitindo que o usuário possa interagir de forma eficiente com o site.

Estrutura básica do HTML

  • html: abre e fecha o código.
  • head: pré carrega informações antes de mostrar o conteúdo para o usuário.
  • body: é o conteúdo mostrado para o usuário.
  • h1: diz respeito ao título do texto.
  • h2-h6: referem-se aos subtítulos do texto.
  • p: adiciona parágrafo ao texto.
  • u: sublinha palavras e frases do texto.
  • b: adiciona o negrito ao conteúdo selecionado, sem valor semântico.
  • strong: negrito semântico,adiciona um ar de urgência e atenção ao conteúdo selecionado.
  • i: diz respeito ao itálico de palavras e frases do texto.
  • blockquote: diz respeito a uma citação para se adicionar no texto.
  • ol: diz respeito de uma lista ordenada, com marcadores numéricos (Tag: PAI).
  • ul: diz respeito de uma lista não ordenada, ao invés de marcadores numéricos, serão 'bolinhas' (Tag: PAI).
  • li: diz respeito a criação de itens da lista (Tag: FILHA).
  • a: faz uma âncora em algum local específico.
  • href: referencia o link criado.
  • form: cria um formulário.
