CSS

CSS é uma linguagem de estilos, usada para adicionar cores, posicionamentos, altura/largura e muitas outras formatações aos elementos e tags HTML .

De forma simples, podemos usar a analogia de que o css é a "pele" de um site. Ele deixa o site mais agradável visualmente.

O css forma o combo web com outras duas tecnologias, são elas: o HTML e o JAVASCRIPT. Seguindo a analogia de que o HTML é o esqueleto, o CSS3 é a pele e o JAVASCRIPT os músculos e orgãos do corpo.

    Cada um com a sua própria função:

  • html: usado para criar a estrutura do site.
  • css: adiciona cores, fontes de texto, espaçamentos, posicionamentos e etc...
  • javascript: adiciona interatividade e movimento, permitindo que o usuário possa interagir de forma eficiente com o site.
